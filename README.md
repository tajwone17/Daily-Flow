# Daily Flow

[![Live Demo](https://img.shields.io/badge/Live-Demo-blue)](https://daily-flow-ruddy.vercel.app/)

Daily Flow is a **personal productivity and task management application** designed to help users organize their daily activities, track progress, and improve workflow efficiency. With an intuitive interface and customizable features, Daily Flow ensures you stay on top of your tasks every day.

## Features

- Create, edit, and delete daily tasks
- Mark tasks as completed
- Categorize tasks by priority or type
- View daily, weekly, or monthly progress
- Search and filter tasks easily
- Responsive design for desktop and mobile
- Optional notifications/reminders
- Collaborative features (if implemented)

## Tech Stack

- **Frontend:**  Next.js
- **Backend:**NextJS
- **Database:** MongoDB 
- **Authentication:** JWT 

## Live Demo

Check out the live project here: [Daily Flow Live](https://daily-flow-ruddy.vercel.app/)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/daily-flow.git
````

2. Navigate to the project directory:

```bash
cd daily-flow
```

3. Install dependencies:

```bash
npm install
```

4. Configure environment variables:

```bash
# Create a .env file
# Add variables like DB connection, PORT, JWT_SECRET, etc.
```

5. Start the application:

```bash
npm start
```

6. Open the app in your browser at `http://localhost:3000` (or your configured port).

## Usage

* Add tasks for today or future dates.
* Mark tasks as done to track progress.
* Use filters and search to manage large task lists.
* Check your dashboard for daily/weekly productivity insights.
